Программа для конвертирования набора файлов в заданном формате и записи содержимого файл, имеющий стандартную структуру - resources.xml
Для реализации данной задачи использован класс XMLConvertion, определяющий общие функции работы с файлами
Для парсинга каждого отдельного набора файлов определяется свой класс, наследующий XMLConvertion
Скрипт запускается из консоли, в качестве дополнительных параметров принимает на вход:
	1)абсолютный путь к папке, где хранится набор директорий с файлами
	2)название класса, определяющего функцию парсинга файлов
